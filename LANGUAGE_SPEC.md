# AuriCode Language Specification

Version: v0.1.0

## Overview

AuriCode is a simple experimental programming language designed to be easy to read and learn.

File extension:

.auri

---

## Variables

Variables store values.

Example:

name = "AZ1255"
score = 100

---

## Output

The "print" statement displays text.

Example:

print "Hello, World!"
print name

---

## Comments

Comments are ignored by the interpreter.

Example:

#This is a comment

---

## Data Types

String

name = "Aryan"

Number

score = 100

Boolean

active = true

---

## Conditions

Conditions execute code when a condition is true.

Example:

if score > 50, then:
print "Passed"

---

## Equality

Example:

if name = "Aryan", then:
print "Welcome"

---

## Features

- Functions
- Loops
- User input
- Modules
- Error handling
- Standard library
