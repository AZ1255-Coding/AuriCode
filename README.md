# AuriCode
AuriCode is a programming language project created by **AZ1255-Coding**.
## Goals
* Simple syntax
* Easy to learn
* Readable code
* Fun to experiment with
## File Extension
**.auri**

For a simple ***tennis*** service RLG (Random Luck Generator),

```AuriCode
firstserve= missile or net

if firstserve=net, then:
    print "First fault!"
    else don't print

secondserve= right or net

if secondserve=net, then:
    print "Double fault!"
    else don't print```
